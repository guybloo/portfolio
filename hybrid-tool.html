Hybrid Tool

A tool for tying palm tree brances

My parents are date growers farmers and our fruits are super juicy

Having such big fruits causes heavy branches which may break

To prevent it, we are tying the branches together to gain more strengh

We can cut one stick from the branch with the knife in the bottom of the tool

When it is open, we can be close it on the branches until it is tightened

From the hole in the main loop we can push the stick until it is around the branches and tie it

This tool in an extension to the hand which allow reaching high branches

It uses sticks instead of plastic, which is a waste of around 800KG a year
